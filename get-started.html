Quick Start Guide

Functions

We have 2 functions files -

  • functions.php: Contains scrips & enqueueing, sidebars, custom post type includes, and everything else you'd normally expect to modify.
  • /includes/base-functions.php: Everything useful that you probably don't need to modify on a project-by-project basis.


Template Structure

Post formats are rarely used in client websites, so we've removed support for them by default. If you would like to use post formats on a project, the method used in underscores (_s) could easily be integrated.

Our recommendation is to use the structure tmpl-templatename.php for custom templates to avoid namespace issues and create easily-accessible body classes for styling.

Reference: WordPress Template Hierarchy


Styles

All of your styles are in the /scss/ folder.

  • style.scss: This is a setup file that builds your stylesheet.
  • /includes/_variables.scss: Set all your colors, fonts, and other variables here. Default grid settings are also here.
  • /scss/responsive-styles/: This folder contains your main stylesheets. These load in according to your default settings from /includes/_variables.scss
    • _xs.scss: The majority of your styles go here. This sheet is loaded on all devices by default.
    • _sm.scss: Small devices (tablets) with a min-width of 768px.
    • _md.scss: Medium devices (desktops) with a min-width of 992px.
    • _lg.scss: Large devices (desktops) with a min-width of 1200px.

Getting Started with SASS + Compass

We're using SASS by default to utilize it's organization, speed, and functions. If you've used LESS before, SASS is an easy transition. If you haven't used CSS preprocessors before, you're in for a treat. You can learn more about SASS here.

Compass is like an extension to SASS, giving us access to all sorts of predefined tools. On the most basic level, Compass can generate cross-browser code for CSS3 properties, saving valuable time and keeping your source code clean.

If you haven't used these tools before, the best way to get started is to visit http://compass-style.org/install/ to setup your compiler, or use an app to handle it for you:

  • CodeKit
  • Compass.ass
  • Scout
